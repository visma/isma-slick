Controls :
- Move                  : UP, DOWN, LEFT, RIGHT
- Pause/Enter menu      : ENTER
- Leave menu/game       : ESC
- Switch Fullscreen     : F

Highscore saving :
- Choose current character          : UP, DOWN
- Choose previous/next character    : LEFT, RIGHT
- Confirm saving                    : ENTER
